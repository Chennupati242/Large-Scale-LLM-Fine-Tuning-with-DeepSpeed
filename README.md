# Large-Scale-LLM-Fine-Tuning-with-DeepSpeed
Enterprise-grade LLM fine-tuning pipeline built with PyTorch, Hugging Face, and DeepSpeed. Leverages ZeRO-2 optimization with CPU offloading and mixed precision (fp16) training to reduce GPU VRAM usage by up to 60%. Enables scalable fine-tuning of billion-parameter models on resource-constrained setups.
